# Java Chat Server & Client

## Описание

Это пример реализации многопоточного чат-сервера и клиента на Java. Проект предназначен для демонстрации работы сетевого
взаимодействия, потоков и логирования сообщений.

## Что делает проект

- Запускает сервер, слушающий указанный порт.
- Позволяет подключаться нескольким клиентам.
- Передает сообщения между клиентами в режиме реального времени.
- Логирует все сообщения в файл `file.log`.
- Поддерживает команду `/exit` для выхода.

## Как запустить

### Предварительные условия

- Java 8 или выше
- В файле `settings.txt` должнн быть указан параметр  `port`
